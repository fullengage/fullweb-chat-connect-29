# Product Overview

This is a SaaS customer service platform built as a Chatwoot-based solution. The system provides comprehensive conversation management, agent assignment, contact handling, and analytics through a modern web interface.

## Core Features

- **Conversation Management**: Kanban-style board for managing customer conversations across different statuses (Open, Pending, Resolved)
- **Multi-tenant Architecture**: Complete isolation between different company accounts
- **Agent Management**: Role-based access control (Agent, Supervisor, Admin) with conversation assignment
- **Real-time Chat**: Full chat interface with support for attachments, private notes, and labels
- **Analytics Dashboard**: Performance metrics and team productivity insights
- **Chatbot Integration**: Automated responses with n8n webhook integration

## Target Users

- **Customer Service Teams**: Primary users managing daily conversations
- **Supervisors**: Team leads overseeing conversation distribution and performance
- **Administrators**: System managers configuring accounts and permissions
- **Business Owners**: Decision makers viewing analytics and metrics

## Business Model

Subscription-based SaaS sold to companies requiring customer service management. Each company gets isolated access to their conversations, agents, and data.
